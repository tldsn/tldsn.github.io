    搜索语法

    • n..n　　

    • n..*

    • >YYYY-MM-DD

    • a not b　　包含a而不包含b　　

    • -n　　排除n

    在GitHub上搜索

    base

    • in:name　　按仓库名称搜索
    • in:description　　按说明
    • in:readme　　按自述
    • user:USERNAME
    • followers:n　　按关注者数量
    • forks:n　　按复刻数量
    • stars:n　　按星号数量
    • created:YYYY-MM-DD
    • pushed:YYYY-MM-DD　　上次更新时间
    • language:LANGUAGE　　按语言搜索
    • topic:TOPIC　　按主题搜索
    • label:LABEL　　搜标签

    搜代码

    • extension:EXTENSION　　限定文件扩展名
    • filename:FILENAME　　限定文件名
    • path:DIRECTORY　　在路径下搜索
    • language:LANGUAGE　　在特定语言下搜索
    • user:USERNAME　　
    • in:file　　文件内容中包含
    • in:path　　文件路径中包含

    键盘快捷键

    在 GitHub 中输入 ? 可弹出一个对话框，列出可用于该页面的键盘快捷键。

    搜索优秀项目:
    awesome + 关键字

    寻找脚手架：
    技术栈 + boilerplate 或者 starter 等关键词
