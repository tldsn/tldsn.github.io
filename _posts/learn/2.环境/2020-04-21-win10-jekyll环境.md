### win10//jekyll

> [http://jekyllcn.com/docs/quickstart/](http://jekyllcn.com/docs/quickstart/)

* 安装ruby
* 安装rubygems
* 安装bundle

```
bundle init

bundle config set path 'vendor/bundle'

bundle add jekyle

bundle exec jekyll new --force --skip-bundle .

bundle install

bundle exec jekyll serve
```
